## RoboMaster机甲大师青少年挑战赛开源项目
### 精准高速移动
这些是为 RoboMaster EP 编写的程序。如要查看更多信息，请参考主程序中的注释。
使用说明：
1.打开RoboMaster APP;
2.点击「实验室」;
3.点击「我的程序」;
4.点击上方导航栏「Python」;
5.新建程序，并将本项目仓库中的「Precise and high-speed movement.py」文件中的代码复制到刚才新建的程序中;
